# 📊 Sweden Unemployment Analysis

## 📌 Overview
This project analyzes Swedish unemployment trends using real data 
from SCB (Statistics Sweden) API. We compare general unemployment 
vs youth unemployment between 2010-2025.

## 🔍 Key Findings
- General unemployment average: 7.8%
- Youth unemployment average: 21.6%
- Youth unemployment is **2.8x higher** than general!
- Both peaked in 2021 due to COVID-19
- Youth unemployment improving 2015-2018, then COVID hit hard

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- SCB PxWeb API (real government data)

## 📊 Data Source
Statistics Sweden (SCB) — Official Swedish government statistics  
https://www.scb.se
